Keeper League Upgrade - Next Steps
Hide Message

Thanks for taking the time to set up your 2019 league home page on MyFantasyLeague.com! This important message will detail the next steps you need to perform to make sure you're ready for the 2019 season. To see a list of improvements and changes for 2019, check out our Site News page.

Most importantly, all of your league rules, including all scoring rules and all waiver rules, are automatically carried over from year-to-year. So, unless your rules have changed since last season, we strongly discourage you from making any un-needed edits to your league rules.

If your league is a keeper (or dynasty) league, whereby some or all players are kept on rosters from year to year, please read on. If your league is not a keeper/dynasty league, you can safely ignore the rest of these instructions. Next steps in this upgrade process include:

  1. Transfer some or all 2018 roster players over to this 2019 league on the Transfer Rosters page.
  2. Set up your "base" 2019 draft order on the Draft Order Setup - All Rounds page. Then use the Draft Order Setup One Round to add any additional picks or remove picks as needed (not pertaining to traded draft picks)
  3. Import 2019 future draft pick trades on the Import Future Draft Picks page.
  4. Depending on your league rules, owners may not need to declare their keeper players until later in the off-season. You can use the Keepers Setup page to indicate how many players owners can keep, and set up an event for owners to select their keepers on the League Calendar. The owners will use the For Owners > Select Keepers page to select their keepers. After the keeper deadline has passed the commissioner can process these on the Load Keepers page. See the Help Center - Commissioner Tutorials for additional details.
  5. If your league tracks contracts, go to the Salary Cap Setup page to reduce the contracts of all players by 1.
